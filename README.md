# E-Commerce Pricing Analysis & Optimization

This final project for my PSTAT135 Big Data Analytics course was done in Google Cloud Platform and analyzes 181,873 order lines across 29,049 SKUs (unique products) and 26 product categories (`ecommerce_data.csv`) to support ecommerce pricing and demand decisions. The notebook combines exploratory analysis with K-Means and PCA product segmentation, monthly category demand forecasting (linear regression with lag and time trend), and a margin-demand priority matrix that ranks categories for protection, repricing, or cost review. The approach supports an integrated workflow over a single predictive score.
