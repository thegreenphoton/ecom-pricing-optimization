# E-Commerce Pricing Analysis & Optimization

This project analyzes ecommerce transaction data to support pricing and demand decisions using exploratory analysis, unsupervised learning, and supervised forecasting. The dataset (BigQuery's theLook E-Commerce dataset) contains **181,873 order lines** across **29,049 SKUs** and **26 product categories**, with fields for price, cost, margin, category, and customer geography.

The notebook begins with exploratory summaries of revenue, profit, and price distributions by product and category. Unsupervised methods segment the catalog using **K-Means** (four clusters on standardized price, cost, margin, and units sold) and **PCA** at both the SKU and order-line levels to visualize how products and transactions vary across pricing and demographic dimensions.

Supervised models focus on **monthly category demand forecasting**. Gradient Boosting with lag features serves as a benchmark (R² ≈ 0.46), while **linear regression** using lag-1 demand, a time trend, and their interaction provides the preferred forecast (R² ≈ 0.96) under a time-based train/test split. Rather than predicting profit directly, which is largely determined by price and cost, the pricing section builds a **margin–demand priority matrix** that classifies categories into protect, reprice, premium niche, or cost-review strategies, and flags SKUs below their category median margin for repricing.
